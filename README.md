# Проект: Разные модели и стратегии на разных участках прогнозного горизонта
 
**Автор:** Аванесян Алина Армавиковна  

## Гипотеза

Разные модели и разные стратегии прогнозирования конкурентоспособны на разных участках прогнозного горизонта. Выбор лучшей модели зависит от шага h.

## Данные

M4 Monthly — 150 случайных рядов (seed=42), горизонт h=18.

## Модели

| Модель | Тип | Стратегия |
|--------|-----|-----------|
| Naive | Бейзлайн | — |
| SeasonalNaive | Бейзлайн | — |
| AutoTheta | Бейзлайн | — |
| AutoETS | Бейзлайн | — |
| CatBoost | ML (глобальная) | MIMO, Recursive, Direct |
| PatchTST | DL (глобальная) | MIMO |

## Структура репозитория

```
├── README.md                  # описание проекта
├── requirements.txt           # зависимости
├── config.py                  # параметры эксперимента
├── run_experiment.py          # запуск пайплайна
├── src/
│   ├── data.py                # загрузка и подготовка данных
│   ├── features.py            # признаки и split
│   ├── metrics.py             # sMAPE, MASE
│   ├── baselines.py           # статистические бейзлайны
│   ├── catboost_models.py     # CatBoost: MIMO, Recursive, Direct
│   └── patchtst_model.py      # PatchTST
└── results/
    ├── analysis_results.ipynb # визуализация и анализ
    └── report.pdf             # итоговый отчёт
```

Вообще исходно весь код я написала в analysis_results.ipynb, а затем разбила его по блокам, поэтому в тетрадке можно увидеть весь пайплайн с комментариями и визуализациями.

## Запуск

```bash
pip install -r requirements.txt
python run_experiment.py
```

Результаты сохраняются в `results/`. Ноутбук `results/analysis_results.ipynb` содержит визуализацию и графики.

## Основные результаты

| Модель | sMAPE | MASE |
|--------|-------|------|
| AutoTheta | 14.15 | 0.94 |
| AutoETS | 14.62 | 0.91 |
| CatBoost Recursive | 16.49 | 1.27 |
| CatBoost Direct | 16.60 | 1.31 |
| PatchTST MIMO | 16.93 | 1.16 |
| Naive | 17.06 | 1.18 |
| CatBoost MIMO | 17.53 | 1.48 |
| SeasonalNaive | 17.54 | 1.19 |

Гипотеза **частично подтверждена**: стратегии CatBoost (Recursive, Direct, MIMO) ведут себя по-разному на разных шагах горизонта, но статистические бейзлайны (AutoTheta, AutoETS) побеждают на всех шагах.

## Метрики

- **sMAPE** — основная метрика
- **MASE** — метрика, сглаживающая различия в масштабе рядов (MASE < 1 = лучше SeasonalNaive)

Обе метрики вычислены в агрегате и в разрезе каждого шага h=1..18.
